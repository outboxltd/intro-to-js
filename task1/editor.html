תרגילים ב-JavaScript

תרגילים:

תרגיל 1: שינוי טקסט בלחיצת כפתור

משימה:

צרו דף HTML עם:

  • כותרת (h1) עם id="welcomeMessage"
  • כפתור שבלחיצה עליו ישנה את תוכן הכותרת
  • פונקציית JavaScript שתשנה את תוכן הכותרת ל"תודה שלחצת על הכפתור!"

ברוכים הבאים לאתר שלי!

תרגיל 2: שינוי צבע רקע

משימה:

צרו דף HTML שיכלול:

  • כותרת מתאימה
  • כפתור "החלף צבע רקע"
  • פונקציית JavaScript שתשנה את צבע הרקע של העמוד ל-lightgreen

משחק החלפת צבע רקע

תרגיל 3: צבע רקע אקראי

משימה:

שדרגו את התרגיל הקודם:

  • צרו מערך של צבעים
  • בכל לחיצה על הכפתור, בחרו צבע אקראי מהמערך
  • שנו את צבע הרקע לצבע שנבחר
